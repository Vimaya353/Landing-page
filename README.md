# Landing Page

A responsive landing page built as part of **The Odin Project Foundations Course**. This project focuses on applying HTML, CSS, and Flexbox concepts to recreate a provided design mockup.

## Preview

This project includes:

- Hero section with navigation bar
- Information cards section
- Testimonial/quote section
- Call-to-action section
- Footer

## Built With

- HTML5
- CSS3
- Flexbox
- Git & GitHub

## Features

- Clean and modern layout
- Flexbox-based positioning
- Consistent typography and color scheme
- Structured and semantic HTML
- Organized CSS styling

## Project Goals

The primary objective of this project was to:

- Practice creating a webpage from a design specification
- Improve Flexbox layout skills
- Learn proper project structure
- Gain experience using Git and GitHub for version control
